# Heart [![Build Status](https://travis.ibm.com/heart/heart-graphql.svg?token=fzzzzZdVsvEKkRSKEdP2&branch=master)](https://travis.ibm.com/heart/heart-graphql)

To start your Phoenix app:

- Install dependencies with `mix deps.get`
- Setup all related services by running `make up`
- Create and migrate your database with `make migrate`
- Start Phoenix endpoint with `mix phoenix.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
